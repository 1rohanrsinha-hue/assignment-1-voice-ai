# Assignment 1 — Real-Time Voice AI Assistant

A local conversational AI assistant built using Faster-Whisper, Ollama, and Kokoro TTS.

## Features
- Speech-to-text using Faster-Whisper
- Local LLM responses using Ollama
- AI voice output using Kokoro
- Real-time microphone interaction
- GPU acceleration with RTX 4060

## Technologies Used
- Python
- Faster-Whisper
- Ollama
- Kokoro
- PyTorch
- FFmpeg

## Pipeline
Mic → Whisper → LLM → Kokoro Voice Output
